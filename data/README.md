# Data Workspace

Local datasets live here and are ignored by git.

- `raw/bi/`, `raw/ji/`: original source videos
- `presenters/bi/`, `presenters/ji/`: renamed presenter archives used by scripts
- `bi_training/`: local BI training workspace
- `bi_training/clips/`: selected 25fps clips with aligned audio
- `bi_training/flame_params/`: VHAP-tracked FLAME coefficients per clip
- `bi_training/lmdb/`: prepared LMDB datasets, train/val splits, and stats
